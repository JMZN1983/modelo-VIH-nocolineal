# Modelo VIH con dos ruidos no colineales — Capítulo 4

Códigos en R para las simulaciones de extinción, persistencia y umbral del modelo VIH.
Incluye comparación determinista vs estocástica (RS, Ding, No-colineal).

## Archivos principales
- `scripts/cap4_vih_nocol_figuras.R`: simulación Euler–Maruyama y densidad estacionaria.
- `scripts/cap4_comparativa_R0.R`: comparación de trayectorias RS–Ding–No-col.
- `run_all.R`: ejecuta ambos scripts.
